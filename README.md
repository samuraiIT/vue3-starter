## Installation

```bash
 #NPM
  npm install vue3-starter

 #YARN
  yarn add vue3-starter
```

### Install depedencies

```bash
#NPM
  npm install

#YARN
  yarn
```

### Start the dev server

```bash
#NPM
  npm run dev

#Yarn
  yarn dev
```

### Build for production

```bash
#NPM
  npm run build

#Yarn
  yarn build
```

### Locally preview production build

```bash
#NPM
  npm run serve

#Yarn
  yarn serve
```
