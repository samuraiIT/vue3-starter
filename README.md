A Vue3 starter, powered by Vite and using Tailwind, I18n and Vue-router.

## [Demo](https://vue3-starter.netlify.app)



## Installation

```sh
npx degit maximedaraize/vue3-starter app
cd app

```

### NPM

```sh
  npm i
  npm run dev
```

### YARM

```sh
  yarn
  yarn dev
```
